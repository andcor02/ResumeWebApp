Hardware Portfolio

Feel free to see a handful of hardware projects I have designed and shown at external conferences such as Embedded World and Mobile World Congress.

Using EDA tools I designed an IoT development board with multiple environmental sensors and WiFi connectivity. This was used in a Mbed C++ cloud workshop with 50 attendees to show how to program IoT devices and transfer the data to a cloud platform to manage the data. Shown at Arm Techcon 2017.
Using EDA tools I designed a LoRa based communications Arduino board to communicate to LoRa gateways and transmit information via a chip antenna at a range of 20km.
Using EDA tools I designed and programmed a 6loWPAN communications environmental sensor platform, the board transmits ultrasonic distance, temperature, noise and acceleration. Shown at Arm APM 2015.
Using 3d CAD design and printing I built a large scale handcrank machine that drives a dc motor to capture analytics on power generation to a microcontroller platform. Shown at Hannover Messe 2018.
Train track model demo to show asset tracking of goods using RFID sensors, model assets and a full running train with live web app. Shown at Hannover Messe 2019.
Environmental sensor demo, various Arm microcontroller platforms communicating to cloud and displaying various environmental data over various communication protocols. Displayed with designed webapp. Shown at Embedded World 2018.
IoT smart city. Designed and developed of a large scale physical smart city model, which was used to educate IoT best practices through interactive stories. Also designed accompanying webapp on each of the touchscreens to control the city. Shown at Mobile World Congress 2018.
Designed small scale conveyor belt system to show the use of Arm microcontrollers in industry 4.0. Constructed the hardware and used PLC (programmable logic controller) to create an autonomous bearing sorting system. Shown at Arm APM 2019.
Designed and programmed PLC board to control conveyor belt system. Shown at Arm APM 2019.
